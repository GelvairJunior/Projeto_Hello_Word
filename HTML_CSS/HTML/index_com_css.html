Testando as cores e degrade nesta Div e explicando algumas CSS

Vamos explicar algumas regras das css

  • para voce criar uma CSS você tem 3 opções fazer na prorpia tag, fazer no head as estilização e o mais recomendado fazer em um arquivo separado e só fazer um link para o HTML

  • primeiro quando selecionar alguma tag para estiliza ira estilizar todas com aquela tag

  • caso queira voce pode adicionar uma class para a tag para modificar apenas aqueles com a class

  • caso queira modificar a cor de fundo use o background color e caso for so a letra o color

  • Voce pode fazer degrade usando o background-image: linear-gradient depois escolha em que direção vai ser o degrade e por fim as cores

  • defina a fonte usando o font family e o tamanho da fonte com font size e defina o tamanho de acordo com o que preferir, ex: 5px

  • se quiser deixar centralizado ultilize just content: center, e caso quiser da direita para esquerda, troque o center por right

  • com o margin você define a distancia que deve ficar cada item

  • o * e para dizer que tudo sera alterado de acordo com a regra que voce definir aqui

Seletores personalizados

Aqui iremos usar o "id"

voce pode ultilizar o id para poder colocar estilização na mesma tag com diferentes estilizações, so usaremos um id em todo o html se quisermos usar a mesma estilização em outra tag use o class, ele é definido pelo #"seuid" ex: #seletor

aqui iremos ultilizar o "class"

Nos iremos ultilizar a class para usar a mesma estilização em diferentes tags mas ainda separar do resto, para identificar o class no css usaremos o .suaclass ex: .class

Também voce consegue usar 2 class, voce ira dar um espaço dentro da class, tambem consegue usar um class e um id na mesma tag porem o id ira sobrepor

Agora iremos ver as Pseudo-classes

aqui vimos que quando voce passa o mouse em cima fca amarelo, as pseudo-classes são para um elemento, class ou id, fassam algo caso aconteça algo, sao focadas no estado do elemento, são identificados pela tag e depois 2 pontos. ex: p:hover

voce pode ver que é possivel mudar algo dentro de elemento usando alguma o faz uma div > p e voce pode mudar algo da tag p que esteja dentro dessa div

OI! você encontrou um ester-egg

já passou o mouse na div a cima?

vamos fazer uma lista das pseudo-classes

  • Hover: quando passar o mouse algo acontece
  • Visited: quando ja tiver visitado algo aconteçe
  • active: quando clicar algo acontece

Pseudo-Elementos

voce consegue adicionar coisas depois ou antes das tags html sem mexer nelas, são identicados por 4 pontos. ex: p::after

  • # = id
  • . = class
  • : = pseudo-classes
  • :: = pseudo-elementos
  • > = childrem "filho"

Modelos de Caixas

Box-level

Paragrafo e h1 - h6 são exemplo de uma caixa Box-level.

O box level sempre ocupa a proxima linha, o box Level ocupa 100% da largura da tela, e depois ele pula para a linha de
(DIV) (h1-h6)(main)(herder)(form)(video)

as box-level podem ficar na mesmas linhas se forem colocada como display inline-block ai configurem da forma que preferir

Brinque aqui com o dev-tool - Box-level

Brinque com o dev-tool para ver exatamente como funciona e faça testes para ver o visual, nao se preocupe nao ira alterar no arquivo.
Para acessar o dev-tool clique com o botão direito do mouse, e clique em "Inspecionar"

Inline-level

tags que ficam no meio do p como o links são exemplos de caixas Inline-level

exemplos: (span) (a)(code)(lavel)(button)(input)(select)
