Variaves do JavaScript

Uma variavel é um valor dentro de algo, no JavaScript temos 4 tipos de variaveis "geralmente é quase todas as linguagens tem isso de variaveis" que são:

  • String: que são variaveis de textos

  • Booleano: que são verdadeiro ou falso

  • Number: o number ele tem dois valores, o Decimal e o Rel. Real é 5,3 e o Decimal é 5, geralmente ele se comporta altomaticamente, porem em algumas situações voce deverá definir cada um

Uma variavel no JavaScript é escrita assim var x = valor_da_variavel

o "var" é para definir que o que esta por vir será uma variavel. O "x" é nome da variavel. O "=" no JavaScript é considerado que recebe um valor exemplo de cima o "x" esta recebendo o valor_da_variavel. E por ultimo o "valor_da_variavel" que é o valor em si pode ser um numero um booleano ou uma string

No JavaScript você geralmente tem que transformar as variaveis

para transformar uma string em number, você irá digitar number antes do valor da variavel, ex: number(2)

para transformar um number em string, você só digita str antes do varor ex: str(2)

Atribuição

usando o que tem a cima você pode alterar o valores ex: var n = 3. n = n + 5 (8). n+=4 (12).

incremento quando voce quiser adicionar ou diminuir apenas 1 você pode usar var n = 5. n++ . n--

Operadores de atribuição

  • = = a por uma informação (recebe)
  • n+= 4 = significa que é o valor de n mais 4
  • n-= 4 = significa que é o valor de n menor 4
  • n*= 4 = significa que é o valor de n multiplica por 4
  • n/4 = 4 = significa que é o valor de n divide 4
  • n**= 4 = significa que é o valor de n potencia 4
  • n%= 4 = significa que é o valor de n resto 4
  • n++ = adiciona n mais 1
  • n-- = diminui n menos 1
  • ${} = poder colocar variavel no meio do texto sem usar concatenação
  • '"" = para poder usar junto do ${}

Operadors relacionais

  • > = maior que
  • < = menor que
  • >= = maior ou igual
  • <= = menor ou igual
  • == = igual à
  • != = diferente de
  • === = se é identico

os resultados desses operadores sempre volta um booleano

Operadores lógicos

  • ! = negação
  • && = conjunção (e logico) (se não der as duas verdadeiras é falso)(e)
  • || = disjunção (ou lógico) (Basta um ser verdadeiro estou satisfeito ai da verdadeiro)(ou)
